# Fixed Deposit (FD) — Knowledge Base

## Overview
Bank Muscat Fixed Deposits provide guaranteed returns with flexible terms for individual, corporate, and institutional clients. FDs are a core savings product. This category covers FD creation, cancellation, renewal, rate disputes, and interest/profit payment issues.

## Product Details
- **Minimum Deposit**: RO 1,000
- **Currencies**: OMR, USD (1 month to 6 years); GBP, EUR, AED (up to 12 months)
- **Interest Payment Frequency**: Monthly, quarterly, semi-annual, or annual (customer's choice at opening)
- **Auto-Renewal**: Default behaviour at maturity — principal (or principal + interest) automatically renewed at prevailing rate. Customer must submit written request at branch to change maturity instructions.
- **Premature Withdrawal**: Allowed but attracts 1% penalty on the interest rate applicable for the actual period held.
- **Digital Access**: FDs can be opened via mBanking (Requests → Accounts → Open Fixed Deposit). Minimum RO 1,000.

## Indicative Interest Rates (OMR, deposits up to RO 100,000)
- 1 Month: 0.75% p.a.
- 1 Year: 3.25% p.a.
- 3 Years: 3.75% p.a.
- 5 Years: 4.25% p.a.
- *(Rates vary by currency and are subject to change — verify current rates)*

## Common Issues & Root Causes

### Fixed Deposit Not Created / FD Requested but Not Created
- **What it is**: Customer submitted FD application (in-branch or digital) but FD was not opened in T24.
- **Root Cause**: Human Error (branch staff didn't process), System Error (digital request failed), Customer Request (inquiry).
- **Resolution**: Contact Customer; Rectify in T24; process the FD with correct terms.

### Fixed Deposit Not Cancelled
- **What it is**: Customer requested FD premature withdrawal/cancellation but it wasn't actioned.
- **Root Cause**: Human Error (instruction not entered), System Error.
- **Resolution**: Contact Customer; Rectify; process cancellation with applicable penalty calculation.

### FD Auto Renewed Without Customer Request / Fixed Deposit Auto Renewed Without Customer Request
- **What it is**: Customer's FD matured and was automatically renewed when the customer intended to withdraw the funds.
- **Root Cause**: Human Error (maturity instruction not recorded), System Error (auto-renewal despite standing instruction to not renew), Branch Delay (instruction not updated in time).
- **Resolution**: Contact Customer; Rectify — if recently renewed and customer wants funds, process premature withdrawal with penalty waiver discussion if bank error caused the renewal.

### FD Matured Instead of Renewed
- **What it is**: Customer wanted FD to auto-renew but it matured to cash instead.
- **Root Cause**: Human Error or System Error.
- **Resolution**: Contact Customer; Rectify — re-create FD with original terms if agreed with customer.

### Incorrect Rate / Profit Not Paid / Compound Profit Not Paid (Meethaq FDs)
- **What it is**: Interest/profit credited does not match the agreed rate, or periodic interest payments were missed.
- **Root Cause**: Human Error (wrong rate entered at creation), System Error (accrual failure).
- **Resolution**: Contact Customer; Rectify — recalculate correct interest, credit difference.

### Monthly Interest Not Added Back
- **Root Cause**: Human Error or System Error in periodic interest crediting.
- **Resolution**: Contact Customer; Rectify; check T24 accrual records.

### Wrong Interest Payment / Frequency
- **What it is**: Interest is paid at wrong frequency (e.g., quarterly instead of monthly) or wrong amount per payment.
- **Root Cause**: Human Error (wrong frequency selected at opening), System Error.
- **Resolution**: Contact Customer; Rectify; adjust future payments.

### Fixed Deposit Redemption Enquiry
- **Root Cause**: Human Error (misunderstanding of process).
- **Resolution**: Contact Customer; explain redemption/cancellation process and penalties; Rectify.

## Key Operational Facts
- Auto-renewal is the default — customers must explicitly opt out via written branch request.
- Maturity instruction change requires a written request submitted to branch before maturity date.
- Premature withdrawal penalty = 1% of the interest rate for the period held (not 1% of principal).
- FDs opened via mBanking: customer selects tenor and source account; instant processing.
- Meethaq FDs use "Profit" instead of "Interest" in all customer-facing communications.
- FD disputes often surface because customers forget they set auto-renewal — verify T24 maturity instruction record before determining root cause.
