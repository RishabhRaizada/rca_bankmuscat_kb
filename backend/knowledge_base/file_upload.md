# File Upload Related Issues — Knowledge Base

## Overview
This category covers complaints where corporate customers upload payroll/salary files for processing. Bank Muscat processes bulk salary payments for companies via file upload at branch or through corporate online banking. Delays directly cause employees not receiving their salaries on time — a high-urgency issue with financial and legal implications.

## What File Upload Is
Corporate customers (companies) submit payroll files to Bank Muscat for bulk salary processing. The file contains employee account numbers and amounts to be credited. Bank Muscat processes this file and credits employee accounts on the specified date.

## WPS (Wage Protection System)
- Oman's government-mandated payroll system ensuring employees receive salaries on time.
- WPS registration for salary transfers requires the Sanad portal (Ministry of Manpower).
- Branch staff cannot process WPS registrations directly.
- File upload for WPS-registered companies must follow the WPS format and submission timeline.

## Common Issues & Root Causes

### Salary Not Credited / File Not Processed
- **What it is**: Company uploaded the payroll file but employee salaries were not credited on the expected date.
- **Root Cause**: Branch Delay (file received but not processed), Operational Issue (insufficient prioritisation, manual bottleneck).
- **Known pattern**: Salary file processing delays cause immediate financial distress for employees who depend on timely salary receipt. High-urgency complaint — should be resolved same day.
- **Resolution path**: Immediately process pending salary file → confirm credit to employee accounts → contact affected corporate customer with apology and confirmation.

### File Upload Error / File Format Issue
- **Root Cause**: Customer Request (incorrect file format submitted by company), Operational Issue (validation failure).
- **Resolution**: Advise company on correct file format; ask to resubmit; provide file format specifications.

## Key Operational Facts
- **Urgency**: Payroll processing is time-critical. A 1-day delay can violate WPS regulations and contractually agreed salary dates.
- **SLA**: Payroll files should be processed within 2 hours of submission during business hours.
- **WPS consequence**: Companies that fail to pay salaries on time via WPS may face regulatory penalties from the Ministry of Manpower. Bank Muscat's processing failure can expose the corporate customer to regulatory risk.
- **Resolution priority**: File upload complaints should be escalated immediately to the branch manager and processed on priority above routine requests.
- **Proactive action**: After resolving, advise the corporate customer to submit files earlier to provide processing buffer.
